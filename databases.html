Original Artifact

What the project did

The original project used a MongoDB CRUD module and a dashboard to display animal shelter data. It already showed the connection between stored records and a dashboard view, but the query logic and output could be improved.

Enhanced Artifact

What I improved

I improved the CRUD module so it can handle projections, sorting, limits, and cleaner query input. I also improved the dashboard with rescue filters, a breed summary chart, and map logic based on field names instead of fixed positions.

  • Dashboard filtering and data presentation
  • Cleaner connection between stored data and front-end output
