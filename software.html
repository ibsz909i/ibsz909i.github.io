Enhancement One

Software Design and Engineering

This section presents my software enhancement using the CS 360 Sensor Demo app. I selected this artifact because it gave me a good chance to improve both the interface and the code structure in a way that users can notice right away.

Artifact: Sensor Demo Course: CS 360 Platform: Android
Download Artifact Files Download Narrative

Enhancement Focus

  • Clearer layout and user feedback
  • Better organization with a helper class
  • Stronger screen output with live and peak movement values
Original Artifact

What the project did

The original version read accelerometer data and displayed the X, Y, and Z values on screen. It worked, but the screen was simple and the display logic was mixed together in the main activity.

Enhanced Artifact

What I improved

I added a more polished layout, a sensor status message, current movement and peak movement values, and a reset button. I also moved formatting logic into a helper class to make the activity cleaner and easier to maintain.

Skills demonstrated

  • Improving app structure without changing the main purpose of the artifact
  • Designing a better user-facing screen for a mobile app
  • Working with Android activity logic and sensor data

Course outcome alignment

  • Professional communication through the written narrative
  • Implementation skills through application improvement and refactoring
  • Security mindset through safer checks and cleaner handling of app state
